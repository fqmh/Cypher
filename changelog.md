## vX.X — Current release
*Released: <add date>*

The full Cypher feature set as documented in this GitBook:

**Monitoring**
- Multi‑account window monitoring with a live activity log and Disconnect / Training / Combat counters.
- Twelve detections: Combat, Disconnect, Training, Protein Maintain, Merchant, Rift, Raid Base, Boss Spawn, Yakuza Raid Cooldown, S.W.A.T Raid Cooldown, Setsuna Boss Cooldown, Turf.

- **Trials** — auto combat with stamina control, skill weaving, auto block, Zone usage, talent toggles.
- **Shadow Boxing** — solo grind with periodic re‑equip/re‑buy.
- **Roadwork** — Minamoto / Senkaimon routes, speed‑calibrated movement, Speed/Stamina training.
- **Fat** — fast training cycle with slot‑watch safety bail‑out.
- **Inventory Slotter** — design, save, and auto‑apply hotbar presets.

**Quality of life**
- Anti‑AFK, Auto‑Leave (Instant/Normal + evasion), Auto‑Return (auto‑rejoin + server hop), Auto‑Wear.
- FPS cap with focus‑uncap, in‑game overlay HUD, periodic stat check, periodic + combat screenshots, Medal clip (F8).
- Discord webhook alerts with optional mention pings and auto‑delete.
- Custom timers (Yakuza Raid, S.W.A.T Raid, Rift defaults).

**General**
- Warm‑glass interface with nine screens and an emergency stop on **F6**.
- In‑app auto‑updater that preserves your settings and license key.

## How to add a new entry

When you ship an update, copy this block to the top of the page and fill it in:

```markdown
## vX.X — <short title>
*Released: <date>*

### Added
- New feature or option.

### Changed
- Tweaked behavior.

### Fixed
- Bug fixes.
```

Keep entries short and player‑facing — describe what changed for the user, not the code.
