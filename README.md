# Welcome to Cypher

Cypher is a Windows automation suite for your Roblox game — a single warm‑glass app that **watches your accounts**, **reacts to what happens in‑game**, and **runs your grinds for you** while you do something else.

It does three things at once:

- **Monitors** — keeps an eye on every game window you have open and reacts to events (combat, disconnects, training stops, raids, turf, and more) with Discord alerts and automatic actions.
- **Macros** — fully automated grinds for Durability, Trials, Shadow Boxing, Roadwork, Fat, and an Inventory Slotter, with built‑in stamina and food handling.
- **Quality of life** — anti‑AFK, auto‑leave, auto‑rejoin, FPS control, an in‑game overlay, periodic stat checks, timers, and webhook notifications.

> **New here? Start with [Getting Started](getting-started.md).** It walks you from "I just got my key" to "the app is running" in a few minutes.

---

## What makes Cypher different

- **It actually sees the game.** Cypher reads the screen with computer vision — Windows OCR for text events and image matching for visuals — so it reacts to what's really happening, not a fixed script.
- **Real input.** Keys and clicks are sent at the hardware level, so the game treats them like a real player.
- **Multi‑account.** Run and watch several windows at once — perfect for AFK farming across alts.
- **Hands‑off recovery.** Disconnected? Training stopped? Cypher can detect it, alert you, and automatically rejoin.

## Before you start

- **Windows only.** Cypher uses Windows‑specific input and screen APIs. It will not run on Mac or Linux.
- **A license key is required.** Cypher is gated behind a license check on launch. See [Getting Started](getting-started.md).
- **Your game window must be 800 × 600.** Cypher's vision is tuned for that exact window size. Other sizes break detection and clicking. More on this in [Troubleshooting](troubleshooting.md).

> **Panic key:** at any time, press **F6** to immediately stop the running macro. Keep that in your back pocket.
