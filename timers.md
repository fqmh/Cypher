# Timers

The **Timer** tab lets you track cooldowns and reminders — raid timers, event windows, anything you don't want to miss. Each timer counts down on screen and alerts you when it's up.

---

## Adding a timer

1. Open the **Timer** tab.
2. Under **Add Custom Timer**, type a **Name** and a number of **Hours** (default **1**).
3. Click **Add**.

The new timer appears in the list with a live **HH:MM:SS** countdown.

---

## Managing timers

Each timer card shows its name, its duration, and the time remaining, plus:

- **Reset** — restart the countdown from the full duration.
- **Delete (✕)** — remove the timer.

When a timer reaches zero, Cypher alerts you (and can notify Discord if a webhook is configured).

---

## Default timers

Out of the box, Cypher comes with a few handy timers already set up:

| Timer | Duration |
|-------|----------|
| **Yakuza Raid** | 2 hours |
| **S.W.A.T Raid** | 2 hours |
| **Rift** | 30 minutes |

Reset these whenever you run the corresponding event so the countdown reflects your real cooldown. Add your own for anything else you want to track.
